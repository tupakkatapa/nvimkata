# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.2] - 2026-02-22

### Changed

- Replace medal system with 6 grades (A-F)
- Remove auto-bail on too many keystrokes; challenges only end on match or `:w`
- Rework hint system: F1 toggles hint, F1 again for detailed hint
- Replace challenge intro screen with `?` help overlay

### Added

- Incompatible save file detection
- Backward-compatible loading of old save files

## [0.1.1] - 2026-02-22

### Added

- Scores are automatically flagged when challenge content changes
- Yellow `*` markers highlight outdated scores in topic and challenge lists
- Navigation hints shown in footer bars
- Progress display added to challenge picker

## [0.1.0] - 2026-02-22

Initial release.
